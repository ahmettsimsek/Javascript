📜 JavaScript Öğrenme Serüveni (JS Ders Notları)
Bu repo, JavaScript dilinin temelinden ileri seviye konularına kadar olan gelişim sürecimi, aldığım notları ve yaptığım pratik uygulamaları içermektedir. Her klasör, dilin belirli bir özelliğini anlamaya yönelik küçük projeler ve kod blokları barındırır.

🚀 İçerik Rehberi
Klasör yapısı, temel programlama mantığından başlayarak modern asenkron yapıya kadar uzanmaktadır:

🔹 Temel Kavramlar & Mantık
let-const-var: Değişken tanımlama, scope kavramı ve farklar.

Koşullu-ifadeler: if-else, switch-case yapıları.

Döngüler: for-döngüsü ve while-döngüsü ile iterasyon.

Number-String-Metodları: Veri tiplerine özel hazır fonksiyonların kullanımı.

Math-Date-objeleri: Matematiksel işlemler ve tarih yönetimi.

🔹 Fonksiyonlar & Yapılar
fonksiyonlar: Parametreli, parametresiz ve arrow functions.

callback: Fonksiyonları birbirine parametre olarak geçirme mantığı.

dizi-dizi-metodu: map, filter, reduce gibi modern dizi işleme yöntemleri.

🔹 Web & Etkileşim
html-js-bağlantısı: DOM manipülasyonu ve HTML elementlerine erişim.

alert-prompt: Kullanıcıdan basit veri alma ve uyarı verme.

setTimeout-setInterval: Zaman yönetimi ve gecikmeli işlemler.

🔹 İleri Seviye Konular
nesne-tabanlı-programlama: Class, Constructor ve Inheritance (OOP) yapısı.

API-cekmek: Axios ve fetch ile asenkron veri yönetimi (Promise, async/await).

npm-modul-kavramları: package.json yönetimi ve harici kütüphane kurulumu.

new_map_kavramı: Modern JavaScript veri yapıları.

🛠 Kurulum ve Çalıştırma
Projeleri yerel makinenizde çalıştırmak için:

Repoyu klonlayın:
```
Bash
git clone https://github.com/kullanici-adin/repo-adin.git
Gerekli paketleri (nodemon, axios vb.) yüklemek için:
```
```
Bash
npm install
Herhangi bir dersi çalıştırmak için klasöre gidip node index.js veya npm start diyebilirsiniz.
```
